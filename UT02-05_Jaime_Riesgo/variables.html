Creación de Variables en JavaScript

Ejemplos de variables con let, const y var

Las variables en JavaScript se pueden declarar con let, const y var.
let: Se utiliza para declarar variables que pueden cambiar su valor.
const: Se utiliza para declarar constantes que no pueden cambiar su valor.
var: Se utiliza para declarar variables (menos recomendado en ES6+).
