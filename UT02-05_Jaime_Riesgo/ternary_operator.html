Operador Ternario en JavaScript

Ejemplo de un operador condicional ternario

El operador ternario es una forma abreviada de escribir una sentencia condicional if...else. Se compone de tres partes: una condición, una expresión que se evalúa si la condición es verdadera y otra expresión que se evalúa si la condición es falsa.

En el siguiente ejemplo, se evalúa si la variable edad es mayor o igual a 18. Si es verdadero, se asigna el mensaje "Eres mayor de edad." a la variable mensaje; si es falso, se asigna el mensaje "Eres menor de edad.".
