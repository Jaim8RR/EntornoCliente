Ejemplos de Hoisting en JavaScript

Hoisting y diferencias entre var, let y const

El hoisting es un comportamiento en JavaScript donde las declaraciones de variables y funciones se "elevan" al inicio de su contexto de ejecución. Esto significa que las variables y funciones pueden ser utilizadas antes de que se declaren en el código.
